# Openbank Portal

An adaptive biometric login portal designed for seniors - secure, simple, and dignified banking access.

## Features

- **Face ID / Biometric Login** - One-look authentication with no passwords
- **SMS 2FA Fallback** - Traditional SMS code verification for those who prefer it
- **Senior-First UI** - Large text, high contrast, clear navigation
- **Microinteractions** - Smooth animations that provide visual feedback without being overwhelming
- **Accessibility** - WCAG compliant, respects reduced motion preferences

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- Framer Motion (animations)

## Getting Started

```bash
cd openbank-portal
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Design Principles

1. **Simplicity** - One task per screen, no clutter
2. **Security** - Bank-grade protection that doesn't sacrifice usability
3. **Dignity** - Never make users feel inadequate or obsolete
4. **Reliability** - Consistent, predictable behavior
